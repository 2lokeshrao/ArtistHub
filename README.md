# ArtistHub — Digital Portfolio & Booking System

Zero-storage portfolio website for makeup artists, hair stylists, and beauty professionals.

## Features
- Dynamic profile with auto-adaptive text colors
- Instagram/P Pinterest URL auto-resolution
- Services management with pricing
- Achievement & Education showcase
- Calendar-based availability system
- WhatsApp booking automation
- Professional PDF pamphlet generation
- Multi-user with shareable profile links
- 3D effects, glassmorphism, responsive design

## Tech Stack
- HTML5, Tailwind CSS, Vanilla JavaScript
- jsPDF for PDF generation
- noembed API for URL resolution
- localStorage for data persistence

## How to Use
1. Open `index.html` in browser
2. Create account with name + 4-digit PIN
3. Fill your profile details in Dashboard
4. Share your profile link with clients

## Deployment
Works as a single HTML file — deploy on GitHub Pages, Netlify, or any static host.
